Important

This installer will need administrator rights to write to non-user folders (e.g. Program Files). Install UT in folder with write permission or run the installer from an administrator commandline with: java -jar name_of_installer.jar

System Requirements

You must have the following installed:

  • GOAL v. 7426 or higher
  • Unreal Tournament 2004/3
  • Unreal Tournament Patch 4 & TitanPack
  • Java
